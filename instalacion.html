Los pasos que debemos seguir para instalación de la aplicación son los siguientes.


1. Copiar la carpeta PiDirect en el raiz C:\

2. Comprobar que tenemos instalado el IIS o el PWS en el equipo, si no es asi, procedemos a la instalacion

3. Comprobar que tenemos instalado php en el sistema

4. Configurar el IIS correctamente

* En directorio particular pondremos C:\PiDirect

* La página de inicio llamada "index.html" la pondremos como documento predeterminado

* Dar permisos de ejecución a la base de datos C:\PiDirect\banca.mdb

* Para poder ejecutar correctamente el cgi que tenemos en la aplicacion agregaremos una nueva unidad virtual,

llamada cgi y le daremos permisos de ejecución.

5. Configuracion del DSN

* A continuación necesitamos un DSN para conectar con la base de datos, nos vamos a

Inicio -> Panel de Control -> Herramientas Administrativas -> Orígenes de datos (ODBC)

Agregamos el DSN de sistema con el nombre "pidirect", con el driver de Microsoft Access *.mdb,

tenemos que enlazarlo con la BD que esta en la ruta C:\PiDirect\banca.mdb


Con ésto tendremos listo nuestro sitio web.
